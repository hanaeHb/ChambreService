## Installer les packages nécessaires pour l'intégration JWT et sécurité
```bash
pip install Flask-JWT-Extended bcrypt python-dotenv
```
- Flask-JWT-Extended : gérer les JWT (access + refresh tokens) pour l'authentification.
- bcrypt : hasher les mots de passe pour la sécurité.
- python-dotenv : gérer les variables d'environnement via un fichier .env.
- pyjwt et cryptography : pour encoder/décoder les JWT et gérer les clés cryptographiques.
